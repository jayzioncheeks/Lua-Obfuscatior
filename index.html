Overview

XENO Library

A powerful, lightweight UI library for Roblox scripts. Buttons, sliders, dropdowns, keybinds, and smart toast notifications — all in one clean API.

⚡ Get Started Join Discord
🎛️
Rich Elements
Buttons, sliders, dropdowns, switches, text boxes, keybinds and more.
🔔
Smart Notifications
Toast alerts with built-in anti-spam tag system per element.
🔍
Search Bar
Built-in search so users can quickly find any element in the panel.
⌨️
Keybinds
Bind any Roblox KeyCode to trigger callbacks instantly.

⚡ Getting Started

Load XENO at the top of your script with a single loadstring call, then set your GUI name and Discord link before adding elements.

ℹ️ XENO requires HttpGet to be enabled in your executor. The library is fetched from GitHub at runtime.
Luau
-- Load the XENO library
local XENO = loadstring(game:HttpGet('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC'))()

--- Configuration ---
local DiscordInvite = "https://discord.gg/TdaWNwNAMa"
local XENOGUINAME   = "YourNameHere"

wait(6) -- Wait for game to fully load

-- Add your elements below...
⚠️ Always call wait(6) before adding elements. This ensures the Roblox game has fully initialized before your UI renders.

🔍 AddSearchBar

Inserts a search input at the top of the XENO panel. Players can type to filter and quickly locate any element by name. Call this before adding any other elements.

XENO:AddSearchBar() Function

No parameters required. Call it once, immediately after loading XENO and before any other Add* calls.

Luau
XENO:AddSearchBar()

🏷️ AddHeader

Adds a bold visual section heading inside the panel. Use headers to separate groups of features such as "Combat", "Farm", or "Utility".

XENO:AddHeader(title) Function

Purely decorative. No callback or interactivity — just a styled divider with a title label.

ParameterTypeDescription
titlestringThe heading text displayed in the panel
Luau
XENO:AddHeader("NORMAL FEATURES")
XENO:AddHeader("COMBAT")
XENO:AddHeader("UTILITY")

📝 AddLabel

Renders a small informational text line inside the panel. Great for adding context beneath a header or displaying current status.

XENO:AddLabel(text) Function

Displays a muted text string. No interactivity — informational only.

ParameterTypeDescription
textstringThe text content to display
Luau
XENO:AddLabel("Standard interactions without alerts")
XENO:AddLabel("These alert you when used")

🔘 AddButton

Creates a clickable button that fires a callback when pressed. The notify flag controls whether a toast notification pops up on click.

XENO:AddButton(label, callback, notify) Function Notify Support

When clicked, callback is invoked. Set notify to true to show a toast alert confirming the action — useful for actions like teleports, checks, or toggles.

ParameterTypeDescription
labelstringButton text shown in the panel
callbackfunctionCalled when the button is clicked
notifybooleantrue = toast on click; false = silent
Luau
-- Silent button
XENO:AddButton("Print to Console", function()
    print("This is a standard button")
end, false)

-- Button with notification
XENO:AddButton("Secure Check", function()
    wait(0.5)
    -- logic here
end, true)

🔀 AddSwitch

A toggle switch for enabling or disabling features. Ideal for auto-farm, aim assist, ESP, or any persistent on/off state.

XENO:AddSwitch(label, callback, notify) Function Boolean State Notify Support

Renders an on/off toggle. The callback receives the new state (true = on, false = off) every time it's flipped.

ParameterTypeDescription
labelstringSwitch label shown in the panel
callbackfunction(state: boolean)Receives true when on, false when off
notifybooleanWhether to show a notification on each toggle
Luau
-- Silent switch
XENO:AddSwitch("Standard Switch", function(state)
    print("Switch is now:", state)
end, false)

-- Auto-Farm with notification
XENO:AddSwitch("Auto-Farm", function(state)
    if state then
        -- start farming loop
    else
        -- stop farming loop
    end
end, true)

🎚️ AddSlider

A draggable slider for numeric values. Perfect for WalkSpeed, JumpPower, FOV, transparency, or anything on a numeric range.

XENO:AddSlider(label, min, max, default, callback, notify?) Function Notify Support

Renders a slider between min and max, initialized at default. The callback fires in real-time as the user drags. When notify is on, XENO's tag system batches rapid updates to prevent notification spam.

ParameterTypeDescription
labelstringName shown above the slider
minnumberMinimum value
maxnumberMaximum value
defaultnumberStarting value (must be within min–max)
callbackfunction(val: number)Called with current value as slider moves
notifyboolean?Optional. Show notifications on change (anti-spam built in)
Luau
-- WalkSpeed slider (16–100, default 16)
XENO:AddSlider("WalkSpeed", 16, 100, 16, function(val)
    if LocalPlayer.Character and
       LocalPlayer.Character:FindFirstChild("Humanoid") then
        LocalPlayer.Character.Humanoid.WalkSpeed = val
    end
end)

-- Slider with notification (tag system prevents spam)
XENO:AddSlider("Notify Slider", 0, 100, 50, function(val)
    -- realtime logic
end, true)
💡 Sliders with notify = true use a tag system to batch rapid updates — your screen won't be flooded while the user drags.

📋 AddDropdown

A single-select dropdown menu. Ideal for mode selection, teleport lists, aim options, or any mutually exclusive choice set.

XENO:AddDropdown(label, options, default, callback, notify) Function Notify Support

Renders a dropdown with a list of string options. The user picks one; the callback fires with the selected string value.

ParameterTypeDescription
labelstringDropdown label in the panel
optionstable<string>List of selectable string options
defaultstringInitially selected option (must exist in options)
callbackfunction(opt: string)Called with the chosen option string
notifybooleanShow notification on selection change
Luau
-- Teleport list (silent)
XENO:AddDropdown("Teleport List", {"Spawn", "Bank", "Safe Zone"}, "Spawn", function(opt)
    print("Teleporting to:", opt)
end, false)

-- Aim mode selector (with notification)
XENO:AddDropdown("Aim Mode", {"Legit", "Blatant", "Rage"}, "Legit", function(opt)
    -- apply aim mode logic
end, true)

☑️ AddMultiSelectDropdown

Like a dropdown, but lets the user pick multiple options simultaneously. Use this for zones, feature flags, or game modes that aren't mutually exclusive.

XENO:AddMultiSelectDropdown(label, options, defaults, callback, notify) Function Notify Support

The callback fires with a table of all currently selected strings whenever the selection changes. Defaults pre-selects one or more options on load.

ParameterTypeDescription
labelstringDropdown label
optionstable<string>All available options
defaultstable<string>Options pre-selected when the panel loads
callbackfunction(selected: table)Called with all selected option strings as a table
notifybooleanShow notification on selection change
Luau
-- Zone selector (silent, default: Spawn)
XENO:AddMultiSelectDropdown(
    "Select Zones",
    {"Spawn", "Bank", "Safe Zone", "Market", "Arena"},
    {"Spawn"},
    function(selected)
        print("Selected:", table.concat(selected, ", "))
    end,
    false
)

-- Mode selector (with notification)
XENO:AddMultiSelectDropdown(
    "Select Modes",
    {"Legit", "Blatant", "Rage", "Silent"},
    {"Legit"},
    function(selected)
        print("Modes:", table.concat(selected, ", "))
    end,
    true
)

✏️ AddTextBox

A freeform text input field. Use it to let players type usernames, custom values, or any string input your script needs.

XENO:AddTextBox(label, placeholder, default, callback, notify) Function Notify Support

Renders a text input. The callback fires when the user submits the text (on focus lost or pressing Enter). Placeholder text is shown when the field is empty.

ParameterTypeDescription
labelstringLabel shown above the input field
placeholderstringGreyed-out hint text when the field is empty
defaultstringPre-filled value on load (use "" for blank)
callbackfunction(text: string)Called with the entered text on submit
notifybooleanShow notification when text is submitted
Luau
-- Username input (silent)
XENO:AddTextBox(
    "Username",
    "Type a username...",
    "",
    function(text)
        print("User typed:", text)
    end,
    false
)

-- Username input (with notification)
XENO:AddTextBox(
    "Username",
    "Type a username...",
    "",
    function(text)
        print("User typed:", text)
    end,
    true
)

⌨️ AddKeybind

Binds a keyboard key to a callback function. Use this for toggling the UI, activating abilities, or any action that needs a hotkey.

XENO:AddKeybind(label, keyCode, callback, notify) Function Notify Support

Registers a hotkey using a Roblox Enum.KeyCode. The callback fires each time the key is pressed. The keybind label and bound key are displayed in the panel.

ParameterTypeDescription
labelstringDisplay name for this keybind in the panel
keyCodeEnum.KeyCodeThe key to listen for (e.g. Enum.KeyCode.RightControl)
callbackfunctionFired each time the key is pressed
notifybooleanShow a notification when the keybind fires
Luau
-- Toggle UI with Right Control (with notification)
XENO:AddKeybind(
    "Toggle Interface",
    Enum.KeyCode.RightControl,
    function()
        ToggleUI()
    end,
    true
)
ℹ️ Any valid Enum.KeyCode works. Popular choices: RightControl, F6, Insert, Home, End.

🔔 Notification System

Every interactive element in XENO supports an optional notify parameter. When true, a toast notification appears whenever that element is interacted with.

✨ XENO uses a tag system to prevent spam. Elements that fire rapidly (like sliders) batch their notifications so your screen stays clean.
notify: boolean Last Parameter
ValueBehaviorUse When
false Silent — no notification shown Internal updates, print-only debug, high-frequency callbacks
true Toast notification appears on use Mode changes, toggles, keybinds, user-submitted inputs

📦 Full Example Script

A complete working script showing every XENO element, organized into two sections — one without notifications and one with.

Luau

local XENOGUINAME   = "YourNameHere"

wait(6)

XENO:AddSearchBar()

-- 1. NORMAL FEATURES SECTION
XENO:AddHeader("NORMAL FEATURES")
XENO:AddLabel("Standard interactions without alerts")

XENO:AddButton("Print to Console", function()
    print("This is a standard button")
end, false)

XENO:AddSwitch("Standard Switch", function(state)
    print("Switch is now:", state)
end, false)

XENO:AddSlider("WalkSpeed", 16, 100, 16, function(val)
    if LocalPlayer.Character and LocalPlayer.Character:FindFirstChild("Humanoid") then
        LocalPlayer.Character.Humanoid.WalkSpeed = val
    end
end)

XENO:AddDropdown("Teleport List", {"Spawn", "Bank", "Safe Zone"}, "Spawn", function(opt)
    print("Teleporting to:", opt)
end, false)

XENO:AddMultiSelectDropdown("Select Zones", {"Spawn", "Bank", "Safe Zone", "Market", "Arena"}, {"Spawn"}, function(selected)
    print("Selected:", table.concat(selected, ", "))
end, false)

XENO:AddTextBox("Username", "Type a username...", "", function(text)
    print("User typed:", text)
end, false)


-- 2. NOTIFICATION FEATURES SECTION
XENO:AddHeader("NOTIFICATION FEATURES")

XENO:AddButton("Secure Check", function()

end, true)

XENO:AddSwitch("Auto-Farm", function(state)

end, true)

XENO:AddSlider("Notify Slider", 0, 100, 50, function(val)
    -- Realtime notification updates (tag system handles spam)
end, true)

XENO:AddDropdown("Aim Mode", {"Legit", "Blatant", "Rage"}, "Legit", function(opt)
    -- Logic here
end, true)

XENO:AddMultiSelectDropdown("Select Modes", {"Legit", "Blatant", "Rage", "Silent"}, {"Legit"}, function(selected)
    print("Modes selected:", table.concat(selected, ", "))
