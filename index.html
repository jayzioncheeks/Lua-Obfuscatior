Overview

XENO UI Library

A sleek, notification-aware UI library for Roblox. Open source, reliable, and built for production scripts.

\u26a1
One-line Load
Get started instantly with a single loadstring call.
\ud83d\udd14
Notification System
Every element supports optional smart alert notifications.
\ud83d\udd0d
Search Bar
Built-in searchable interface to find elements instantly.
\ud83c\udfa8
Clean Design
Polished, consistent UI across all elements and states.

All Elements

AddButtonClickable action button
AddSwitchToggle on/off state
AddSliderNumeric range input
AddDropdownSingle-select option list
AddMultiSelectDropdownMulti-option selector
AddTextBoxFree text input field
AddKeybindKeyboard shortcut binding
AddHeaderSection heading
AddLabelStatic description text
AddSearchBarEnable search functionality

Quick Example

local XENO = loadstring(game:HttpGet('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC'))()

XENO:AddSearchBar()
XENO:AddHeader("My Script")
XENO:AddButton("Do something", function()
    print("Button pressed!")
end, false)
\ud83d\udca1
Notification parameterThe last parameter on most elements is a boolean \u2014 set it to true to fire a notification when the element is used.
Getting Started

Loading XENO

Load the library into your script with a single line via HttpGet.

Load String

local XENO = loadstring(game:HttpGet(
    'https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC'
))()
\u26a0\ufe0f
Executor SupportRequires an executor with HttpGet support (Synapse X, KRNL, Fluxus, etc.).

Discord & Config

local DiscordInvite = "https://discord.gg/TdaWNwNAMa"
local XENOGUINAME  = "YourNameHere"

Set DiscordInvite to your own server link. Set XENOGUINAME to your script's display name.

Initialization Delay

wait(6)

XENO:AddSearchBar()

A short wait() after loading ensures the game has fully initialized before building the UI.

Basic Setup

Organize your script with sections, labels, and UI elements.

Full Script Template

local XENO = loadstring(game:HttpGet('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC'))()

local DiscordInvite = "https://discord.gg/TdaWNwNAMa"
local XENOGUINAME  = "YourNameHere"

wait(6)

XENO:AddSearchBar()

XENO:AddHeader("NORMAL FEATURES")
XENO:AddLabel("Standard interactions without alerts")

XENO:AddButton("Print to Console", function()
    print("This is a standard button")
end, false)
\u2705
TipUse AddHeader and AddLabel to visually separate sections of your UI for better readability.
Elements

Button

A clickable button that runs a callback when pressed. Supports optional notifications.

XENO:AddButton function

Creates a pressable button element in the UI. Fires the provided callback when clicked.

Parameters
NameTypeDescription
labelstringText displayed on the button
callbackfunctionFunction called when button is pressed
notifybooleanWhether to fire a notification on press

Examples

Without notification

XENO:AddButton("Print to Console", function()
    print("This is a standard button")
end, false)

With notification

XENO:AddButton("Secure Check", function()
    wait(0.5)
    -- your logic here
end, true)
Elements

Switch

A toggle element that holds an on/off boolean state and fires a callback on change.

XENO:AddSwitch function

Creates an on/off toggle. The callback receives the new state as a boolean.

Parameters
NameTypeDescription
labelstringText label for the switch
callbackfunction(state: boolean)Called with new state when toggled
notifybooleanWhether to fire a notification on toggle

Examples

-- Standard switch
XENO:AddSwitch("Standard Switch", function(state)
    print("Switch is now:", state)
end, false)
